O que faz o MySQL ser tao popular e a sua facil adapetacao para engenheiros de software, que nesse banco relacional nao ha tantas diferencas com outros bancos, assim evitando surpresas!
MySQL esta disponivel para todos os sistemas, e caso nao esteja para algum sistema em questao e possivel baixar o seu codigo binario para usalo.

O desenvolvimento do MySQL, sempre foi focado tao em velocidade como tambem outros requesitos cruciais que um BD tem que ter.
MySQL oferece total gerenciamento de seguranca para voce, um dos exemplos sao: Usuario deve ter um SSL  valido para se conectar com BD
